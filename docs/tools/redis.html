# Redis

# Cài

Mac

Install

brew install redis

Start

redis-server /usr/local/etc/redis.conf

Stop

redis-cli shutdown

Ubuntu

Install

$sudo apt-get update 
$sudo apt-get install redis-server

Start

redis-server

Stop

redis-cli shutdown

# Xài với Command Line

redis-cli

Test

redis-cli ping

See all keys

keys
