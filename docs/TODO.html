# Threads vs Process

Main differences between process and thread:

🔹 Processes are usually independent, while threads exist as process subsets. 🔹 Each process has its own memory space. Threads that belong to the same process share the same memory. 🔹 A process is a heavyweight operation. It takes more time to create and terminate. 🔹 Context switching is more expensive between processes. 🔹 Inter-thread communication is faster for threads.
